# Google Business Profile

## Overview

The Google Business Profile (GBP) is the cornerstone of the "GBP-first" approach. This section focuses on complete optimization of the GBP listing based on Google's official recommendations and local search best practices.

## Why GBP Comes Before Website

**The GBP-First Philosophy:**
1. **Immediate Visibility:** GBP appears in local search results before websites
2. **Trust Factor:** Google verification adds credibility
3. **Content Foundation:** GBP work defines messaging for all other platforms
4. **Quick Wins:** Fastest path to improved discoverability
5. **Customer Journey:** Most local searches start with Google Maps

## Purpose

- Achieve 100% profile completion
- Optimize all GBP elements per Google guidelines
- Establish content strategy for ongoing posts
- Implement review response system
- Maximize local search visibility

## Workflow

1. Audit existing GBP (if exists) or claim listing
2. Complete all profile information per checklist
3. Write optimized business description using brand pillars
4. Add comprehensive services section
5. Upload high-quality photos and videos
6. Select appropriate attributes
7. Seed Q&A section with strategic questions
8. Implement review request system
9. Create posts content calendar
10. Final quality check

## Success Criteria

- [ ] Profile shows 100% completion in GBP dashboard
- [ ] Business description incorporates brand pillars
- [ ] All applicable services listed with descriptions
- [ ] Minimum 20 high-quality photos uploaded
- [ ] All relevant attributes selected
- [ ] 5+ Q&A entries with strategic questions answered
- [ ] Review response system in place
- [ ] Posts content calendar created for next 30 days
- [ ] Profile passes Google's quality guidelines

## GBP Ranking Factors

Based on research, Google prioritizes:

1. **Relevance** - How well profile matches search queries
2. **Distance** - Proximity to searcher
3. **Prominence** - Review count, rating, citations, website authority

**What we control:**
- Profile completeness (relevance)
- Accurate categories (relevance)
- Keyword optimization in description and services (relevance)
- Review generation and response (prominence)
- Post frequency (activity signals)
- Photo/video quality and quantity (engagement)
