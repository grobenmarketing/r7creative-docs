# Workflow Guide

## Overview

This document outlines the complete workflow for content creation and client management, including both "You Capture We Create" and full-service production models.

---

## You Capture We Create Workflow

### Overview

The "You Capture We Create" model empowers clients to capture their own raw content while the team handles all editing, optimization, and publishing.

### Process Flow

```
Client Captures Content → Upload to Frame IO → Team Edits & Optimizes →
Client Approves → Team Schedules & Publishes → Performance Monitoring
```

---

### Phase 1: Planning & Preparation

**Team Responsibilities:**
1. Create content briefs based on Brand Pillars and Content Strategy
2. Provide capture guidelines and scripts/talking points
3. Schedule capture sessions with client
4. Set up Frame IO folder structure

**Client Responsibilities:**
1. Review content briefs and talking points
2. Set up capture environment (lighting, quiet space, etc.)
3. Test equipment and internet connection
4. Block time for capture sessions

**Timeline:** 2-3 days before capture

---

### Phase 2: Content Capture (Client-Led)

**Capture Methods:**

**Option A: Self-Recording**
- Client records on their own schedule
- Uses their own device (phone, camera, etc.)
- Follows provided guidelines and talking points
- Uploads raw files to Frame IO

**Option B: Guided Capture Session**
- Team joins via video call to guide client
- Client records while team provides direction
- Real-time feedback and retakes as needed
- Client uploads immediately after session

**Best Practices:**
- Record in a quiet, well-lit space
- Use landscape orientation for video
- Record multiple takes if needed
- Capture 20-30% more content than needed for editing flexibility
- Test equipment before official capture

**Client Deliverable:**
- Raw video/audio files uploaded to Frame IO
- Any additional assets (documents, images, etc.)

**Timeline:** Scheduled capture session or within agreed window

---

### Phase 3: Upload & Organization

**Client Actions:**
1. Upload all raw content to designated Frame IO folder
2. Add notes or context if needed
3. Flag any takes they prefer or want excluded
4. Notify team that content is ready

**Team Actions:**
1. Receive notification of new uploads
2. Download and organize files
3. Review content for quality and usability
4. Flag any issues or need for re-capture

**Timeline:** Upload immediately after capture

---

### Phase 4: Editing & Production

**Team Responsibilities:**

**Video Editing:**
- Select best takes
- Cut and trim for optimal length
- Add graphics, captions, and overlays
- Color correction and audio enhancement
- Add music/sound effects if applicable
- Optimize for each platform

**Copy & Captions:**
- Write compelling captions based on Brand Pillars
- Include relevant hashtags
- Create platform-specific variations
- Add calls-to-action

**Thumbnail Creation:**
- Design eye-catching thumbnails
- Ensure brand consistency
- Optimize for each platform

**Quality Assurance:**
- Review all elements for accuracy
- Check brand alignment
- Verify platform specifications
- Proofread all text

**Timeline:** 3-5 business days from upload

---

### Phase 5: Client Review & Approval

**Team Actions:**
1. Upload edited content to Frame IO review folder
2. Add any notes or context
3. Request client feedback
4. Set review deadline

**Client Actions:**
1. Review all content pieces
2. Provide feedback via Frame IO comments
3. Approve or request revisions
4. Respond within agreed timeframe (typically 24-48 hours)

**Revision Process:**
- Minor edits: 1 business day
- Major revisions: 2-3 business days
- Client gets [X] rounds of revisions included

**Timeline:** Client reviews within 24-48 hours

---

### Phase 6: Scheduling & Publishing

**Team Responsibilities:**
1. Finalize all approved content
2. Schedule posts in publishing tools
3. Optimize posting times for each platform
4. Set up any necessary platform-specific features (polls, tags, etc.)
5. Double-check all scheduled content

**Publishing Platforms:**
- [Platform 1] - [Frequency]
- [Platform 2] - [Frequency]
- [Platform 3] - [Frequency]

**Content Calendar:**
- Maintained in ClickUp
- Client has view access
- Shows all scheduled content
- Updated weekly

**Timeline:** Published according to agreed schedule

---

### Phase 7: Performance Monitoring & Reporting

**Team Responsibilities:**

**Ongoing Monitoring:**
- Track engagement metrics
- Respond to comments (if included in service)
- Monitor performance trends
- Identify top performers

**Regular Reporting:**
- Weekly metrics summary
- Monthly detailed analytics report
- Insights and recommendations
- Content performance comparison

**Optimization:**
- Adjust strategy based on data
- Test new content approaches
- Refine targeting and timing
- Iterate on top performers

**Client Communication:**
- Bi-weekly sync meetings
- Share wins and insights
- Discuss strategy adjustments
- Plan upcoming content

**Timeline:** Ongoing, with formal reviews bi-weekly

## Full-Service Production Workflow

### Overview

For clients who prefer a fully managed approach, the team handles all aspects of content creation from concept to publication.

### Process Differences from "You Capture We Create"

**Content Creation:**
- Team develops all content in-house
- Uses stock footage, graphics, and licensed assets
- Creates written content, graphics, and design work
- No client capture sessions required

**Client Involvement:**
- Strategic input and direction
- Review and approval of content
- Minimal time commitment

**Ideal For:**
- Clients without time for content capture
- Content types that don't require client on camera
- Written content, graphics, and curated posts

## Troubleshooting Common Workflow Issues

### Client Not Uploading Content on Time
**Solutions:**
- Send reminder 2 days before deadline
- Offer to schedule guided capture session
- Adjust content calendar if needed
- Have backup content plan

### Delayed Feedback/Approvals
**Solutions:**
- Set clear deadlines in initial agreement
- Send reminder when review is due
- Escalate to main contact if needed
- Have approval timeout policy in agreement

### Quality Issues with Client-Captured Content
**Solutions:**
- Provide more detailed capture guidelines
- Offer training or guided session
- Adjust workflow to guided capture model
- Discuss upgrade to full-service if persistent

### Scope Creep
**Solutions:**
- Reference service agreement and scope
- Document additional requests
- Offer additional services at appropriate rate
- Clearly communicate what's included vs. add-on
