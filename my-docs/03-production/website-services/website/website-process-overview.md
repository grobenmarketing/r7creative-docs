## **Service Overview**

**Target Audience:** Local service businesses leveraging Google Business Profile traffic

**Core Concept:** Create an optimized website that mirrors and matches the client's Google Business Profile

---

## **Website Activation Types**

R7 Creative offers three activation tiers for client onboarding:

### Standard Onboarding ($300 Founder / $600 Retail)

**For:** Content-only packages (no website services)

**Includes:**
- Client onboarding process
- Brand Strategy Meeting (60-90 minutes)
- Brand Pillars foundation document ($1,500 value)
- ClickUp workspace setup
- Frame IO project setup
- Content Strategy Meeting
- Implementation of You Capture We Create workflow

**Timeline:** 1-2 weeks from contract signing to first content published

**Ideal For:** Clients focusing solely on Google Business Profile and social media content management

---

### Website: LaunchPad ($1,450 Founder / $1,950 Retail)

**For:** Clients who need basic website presence with light maintenance

**Includes Everything in Standard Onboarding PLUS:**
- Basic website setup (3-5 pages):
  - Homepage
  - About page
  - Services overview
  - Contact page
  - One additional page (testimonials, FAQ, or portfolio)
- Hugo framework implementation
- Mobile-responsive design
- Netlify hosting setup (free tier)
- Custom domain connection
- GBP mirror integration (NAP, hours, service area in footer)
- Contact form with Netlify Forms integration
- Basic SEO optimization
- SSL certificate (automatic via Netlify)
- Two Clocks update system setup

**Ongoing Maintenance:**
- 30 credits/month for website updates via Two Clocks system
- Includes text changes, image swaps, minor content updates

**Timeline:** 3-4 weeks from contract signing to website launch

**Ideal For:**
- Local service businesses needing basic online presence
- Clients with simple service offerings
- Businesses wanting professional site without complexity
- Budget-conscious clients who need "good enough" quickly

**Example Client:** A local plumber who needs a clean, simple website that shows services, contact info, and testimonials. Light updates 2-3 times per month.

---

### Website: Authority Builder ($2,750 Founder / $3,950 Retail)

**For:** Clients who need comprehensive, content-heavy sites with frequent updates

**Includes Everything in LaunchPad PLUS:**
- Comprehensive website (10-15 pages):
  - All LaunchPad pages
  - Individual service pages (4-6 pages)
  - Team page with bios
  - Service area pages
  - Process/How It Works page
  - 2-3 initial blog posts
- Advanced Hugo framework with component library
- Full SEO optimization:
  - Schema markup (LocalBusiness, Service, FAQPage)
  - Meta descriptions and open graph tags
  - Optimized heading structure
  - Internal linking strategy
  - YAML front matter with rich metadata
- Blog system with categories and tags
- Sortable image gallery with category filters
- FAQ accordion system (pulls from data file)
- Advanced contact form with multiple fields
- Newsletter signup integration
- Social proof elements (testimonials, case studies)
- Link-in-bio landing page
- Authority positioning content structure

**Ongoing Maintenance:**
- 60 credits/month for website updates via Two Clocks system
- Includes all LaunchPad updates plus new pages, blog posts, gallery updates

**Timeline:** 5-6 weeks from contract signing to website launch

**Ideal For:**
- Established businesses with multiple services
- Clients focused on content marketing and SEO
- Businesses that need frequent site updates
- Companies positioning as industry authorities
- Clients with visual portfolios (galleries)

**Example Client:** A home remodeling company with 6 service lines (kitchens, bathrooms, additions, etc.), 50+ project photos, team of 5, and weekly blog content. Needs 4-5 updates per month including new projects, blog posts, and service page tweaks.

## **Technical Stack**

- **Static Site Generator:** Hugo
- **Hosting:** Netlify
- **Content Management:** Decap CMS (headless, optional)
- **Version Control:** GitHub (repositories for all code)
- **Domain Registration:** Namecheap or Netlify-native alternative

---

## **Website Structure**

**Page Templates:**
- Homepage template
- Simple/Contact page template
- Inner page template
- Blog folder with taxonomy sorting and author tags
- Pre-made component galleries (HTML/CSS/JavaScript)

**Development Approach:**
- Start with baseline Hugo website with starter pages
- Claude Code (Anthropic's AI) modifies and customizes theme styling

---

## **Information Gathering Process**

**Two Client Interviews:**

1. **Brand Pillar Interview** – Establishes brand strategy, market positioning, target audience
2. **Website Goals Interview** – Defines primary/secondary website goals, copy direction, SEO/AI optimization strategy

**Tool:** Google's Gemini to process and translate interview data

---

## **Content & Optimization Requirements**

**On-Page SEO:**
- Schema markup implementation
- YAML front matter with metadata
- 180-word description per page
- One primary question answered per page
- Author tags and company association for AI knowledge graph building

**Google Business Profile Alignment:**
- Business name, contact info, social media must match exactly
- Service area pages required
- Product/service pages required
- Internal linking between pages and services/products
